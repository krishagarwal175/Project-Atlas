# Glossary

- **Research OS** — this whole system: turns knowledge → understanding → tested theories → high-quality decisions, and remembers why.
- **Evidence Ledger** — append-only table on a claim-bearing note recording supporting/contradicting evidence. The shared primitive.
- **Understanding Layer** — Module 9. Promotes patterns into falsifiable, confidence-tracked **Theories**.
- **Theory** — an explanatory, falsifiable claim ("X causes Y under conditions Z"), never absolute, always evolving.
- **Pattern** — a descriptive regularity across case studies ("these 6 cases share X"). Feeds theories.
- **Contradiction Engine** — Module 10. Actively hunts counterexamples; anti-confirmation-bias is structural.
- **Confidence state** — derived ladder (Speculative→Institutional), never an arbitrary %.
- **Freshness / Decay** — Module 11. Labels knowledge staleness by type half-life. Never deletes.
- **Strategic DNA** — Module 12. Mission/values/non-negotiables that every recommendation is filtered against.
- **Decision Quality** — how good the *reasoning* was, scored at decision time, independent of **Decision Outcome** (what reality did).
- **MOC** — Map of Content; a hub note that links a cluster together.
- **Substrate** — the markdown vault; the source of truth. Everything else is a rebuildable cache.
