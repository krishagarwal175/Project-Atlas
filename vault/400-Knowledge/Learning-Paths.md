# 🎓 Learning Paths

Guided reading sequences through the vault. Each ends by applying the ideas to a **real open question**, not just absorbing them.

## Path A — Think Better (decision science)
For sharpening judgment on any decision.
1. [[MM-decision-quality-vs-outcome]] — the mindset shift
2. [[MM-expected-value]] → [[MM-base-rates]] → [[MM-calibration]] — the quantitative core
3. [[MM-second-order-thinking]] → [[MM-inversion]] → [[MM-systems-thinking]] — see the whole board
4. [[MM-cognitive-biases]] — the traps + countermeasures
5. Apply → score a real decision in the [[F-MCDA-weighted-sum|Decision Engine]].

For Acredemia's core challenge.
