# 🧰 OKRs & the North Star Metric

> **North Star**: the single metric that best captures the value you deliver to customers. **OKRs**: an Objective (qualitative, ambitious) with 3-ish Key Results (measurable) that ladder toward it.

## What it is
A North Star Metric (NSM) aligns the whole company on *delivered value* — not vanity (signups) but the moment of value (e.g. "verified profiles viewed by recruiters," "weekly active teams"). OKRs then set quarterly Objectives with measurable Key Results that move the NSM.

## When to use it
Focus and alignment, especially as you add people. Even solo, an NSM stops you optimizing the wrong number.

## How to run it
1. Pick one NSM that (a) reflects real value and (b) predicts revenue. 2. Set ≤3 Objectives/quarter, each with 2–4 KRs. 3. KRs should be *leading* where possible. 4. Review honestly; grade at quarter end.

## Why it works
A single metric prevents local optimization; OKRs create a testable link between daily work and strategic value.

## Limitations / when NOT to use it
- **Wrong NSM** optimizes the wrong behavior (Goodhart's Law: a metric under pressure stops being a good metric).
- **Sandbagging / cascading bloat** — too many OKRs = no focus.
- Vanity NSMs (signups, page views) mislead; choose an *engaged-value* metric.

## Applies to Acredemia
Candidate NSM: **verified candidates whose profile a recruiter actually engages with, weekly.** It reflects both sides of the marketplace and predicts WTP. See the OKR tracker concept and [[Long-Term-Vision]].
